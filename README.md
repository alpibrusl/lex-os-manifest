# lex-os-manifest

[![CI](https://github.com/alpibrusl/lex-os-manifest/actions/workflows/ci.yml/badge.svg)](https://github.com/alpibrusl/lex-os-manifest/actions/workflows/ci.yml)

**Part of the [Lex](https://lexlang.org) project** — Substrate · [Manifesto](https://lexlang.org/manifesto) · [All packages](https://lexlang.org)

> Manifest format as a Lex package: trust dimensions, grants, narrowing inheritance as types.

## Install

Install the `lex` toolchain: https://github.com/alpibrusl/lex-lang/releases

## Use

Part of the Lex stack. Type-check with `lex check src/*.lex`, run tests with `lex test`, and see `examples/`.
